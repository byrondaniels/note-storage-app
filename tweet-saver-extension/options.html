üí° Pro Tips

  • Test your API endpoint before saving tweets to ensure it works correctly
  • Use the browser's developer tools to monitor network requests if you encounter issues
  • Your API should return a 2xx status code to indicate successful saves
  • Consider adding authentication headers if your API requires them (modify the content script)
  • The extension works with both twitter.com and x.com domains

üîß Troubleshooting

API requests are failing

Common solutions:

  • Check that your API endpoint is accessible and accepts POST requests
  • Ensure CORS headers are properly configured on your server
  • Verify the JSON template is valid and matches your API's expected format
  • Check browser console for detailed error messages
Save buttons are not appearing

Try these steps:

  • Refresh the Twitter/X page
  • Check that the extension is enabled in chrome://extensions/
  • Make sure you're on twitter.com or x.com
  • Check browser console for any JavaScript errors
